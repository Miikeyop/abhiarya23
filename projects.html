Projects

Heart Disease Prediction using Logistic Regression & Random Forest

This project focuses on predicting heart disease using machine learning. Two models were developed: Logistic Regression with 85% accuracy and Random Forest with 98% accuracy. The models analyze features like age, cholesterol, blood pressure, and max heart rate from the UCI Heart Disease dataset to predict disease presence. This work aims to enable early diagnosis and better health outcomes.

Heart Disease Prediction Web App using Streamlit

This project builds a user-friendly web application using Streamlit that allows users to input health data and get instant heart disease predictions based on the logistic regression model. The app presents results clearly with confidence scores and explains key features influencing predictions, making it accessible to everyone.
